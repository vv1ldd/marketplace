# Sovereign disk + Coolify install

**Moved to `vv1ldd/coolify` branch `sovereign`.**

Scripts live in `scripts/sovereign-disk/`. Install via git:

```bash
git clone --depth 1 -b sovereign https://github.com/vv1ldd/coolify.git /tmp/coolify-sovereign
bash /tmp/coolify-sovereign/scripts/install-sovereign.sh
```

This directory keeps a thin `install-meanly-sovereign.sh` wrapper for Meanly docs compatibility.

See `vv1ldd/coolify` → `scripts/sovereign-disk/README.md` and `README-SOVEREIGN-VPS.md`.
